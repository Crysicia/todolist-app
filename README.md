# THP-Todo !

## :wrench: Installation guide
To install and test our awesome project :

    $ git clone https://github.com/Crysicia/todolist-app.git
    $ cd todolist-app
    $ bundle install
    $ rails db:create db:migrate db:seed
  Run it like this :

    $ rails s
    
  Then you can go to :

    $ localhost:3000
